🎯 Como usar essa área

  • Top navbar preta de cada protótipo: menu de demo pra pular entre telas, modais, estados (não faz parte do produto).
  • Interatividade real: botões avançam fluxo, modais abrem/fecham, PIX simula confirmação, cartão recusado é probabilístico.
  • Estilo sketch handwritten: Caveat (títulos) + Patrick Hand (corpo) + paleta paper/ink. Intencionalmente "rascunho" pra convidar mudança.
  • Sally (UX BMAD) pode ser convocada via /bmad-ux-designer pra review de UX antes de fechar uma iteração — ver ux-prototype-workflow.md.
  • Histórico em archive/ — wireframes ASCII antigos, transcrições, decisões originais. Não-canônico (só consulta).
